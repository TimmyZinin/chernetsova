## 1. Позиционирование

**Продукт:** MentalStab — система проактивного мониторинга ментального здоровья для разработчиков, работающих с AI.

**Tagline:** Mission Control для вайб-кодеров

**Миссия:** Мы не дадим тебе сгореть. Наблюдаем за состоянием вайб-кодеров как врачи в ЦУПе — за космонавтами.

**Positioning statement:** MentalStab — единственный продукт, который проактивно мониторит ментальное и физическое состояние IT-специалистов через трекеры, ежедневные чекины и AI-анализ, с эскалацией до живого психолога при критических показателях.

**Модель:** Подписка (recurring) + разовые консультации.

## 2. Целевая аудитория

### Первичная (beachhead)
- Вайб-кодеры — используют Claude Code, Cursor, Copilot для ежедневной работы
- Возраст: 25-40 лет
- Доход: 150K-450K ₽/мес
- Локация: Россия, русскоязычные

### Вторичная (расширение)
- IT-предприниматели, фаундеры, CTO
- Тимлиды с командами вайб-кодеров (B2B2C)

### Боли
- Выгорание (68% разработчиков, Harvard Business Review 2026)
- Потеря сна (ночной кодинг, "ещё один промт")
- Тревожность, FOMO ("AI заменит меня")
- Разрушение отношений ("я просто занят")
- Потеря ощущения контроля над здоровьем

### Мотивация
- Оставаться продуктивным долгосрочно
- Не "сломаться" на пике карьеры
- Объективные данные о своём состоянии (а не субъективные ощущения)

## 3. Конкурентный ландшафт

**Прямых конкурентов: 0.** MentalStab — first mover в нише "проактивный мониторинг ментального здоровья IT-специалистов".

### Конкурентная карта

| Квадрант | Продукты | Слабость vs MentalStab |
|----------|----------|----------------------|
| **High-touch + Tech-specific** | ПУСТО (мы здесь) | — |
| High-touch + General | Yasno, Zigmund, BetterHelp | Реактивная модель (клиент сам записывается). Нет мониторинга, нет данных трекеров |
| Low-touch + Tech-specific | Нет | — |
| Low-touch + General | Calm, Headspace, Wysa, Woebot | Self-service. Нет живого психолога. Нет проактивных интервенций. Woebot ушёл из B2C |

### Ближайший аналог
- **Wane** (getwane.app) — мониторинг burnout через Oura/Apple Watch/WHOOP. Но: приложение (не Telegram), нет русского, нет живого психолога, нет эскалации звонком.

### Конкурентный моат
1. Уникальная экспертиза Екатерины в IFS + контекст разработчиков
2. Данные: чем больше данных о паттернах вайб-кодеров, тем точнее предсказания
3. Сообщество: офлайн-ретриты создают switching cost
4. Проактивная модель vs реактивная — фундаментально другой подход
5. Telegram-нативный (не требует установки приложения)

## 4. Функциональные требования

### 4.1 Ежедневные чекины (MVP)
- Telegram-бот спрашивает: настроение, сон, энергия, стресс (шкала 1-5)
- Свободное поле: "что произошло сегодня?"
- AI-анализ паттернов (скользящее среднее 7 дней, тренды)
- Время чекина: утренний (после пробуждения) + вечерний (перед сном)

### 4.2 Интеграция трекеров здоровья

#### MVP (Sprint 1-2)
| Трекер | Метод интеграции | Стоимость | Данные |
|--------|-----------------|-----------|--------|
| **Oura Ring** | REST API v2 + OAuth 2.0 + Webhooks | $0 | Readiness Score, HRV (ночной, 5-мин), сон (стадии + score), ЧСС (24/7), SpO2, активность |
| **Apple Watch** | Health Auto Export app → webhook на наш backend | $4.99/мес (юзер) | HRV, ЧСС, сон (Core/REM/Deep), SpO2, шаги |

#### Scale (Sprint 3+)
| Трекер | Метод | Стоимость |
|--------|-------|-----------|
| **WHOOP** | Developer API, OAuth 2.0, webhooks v2 | $0 (но WHOOP подписка $30/мес для юзера) |
| **Garmin** | Health API (партнёрская заявка) | $0 |
| **Все трекеры** | Terra API (агрегатор 100+ устройств) | $0 до 200 юзеров, потом $399/мес |

#### Архитектура приёма данных
```
[Apple Watch] → Health Auto Export → POST JSON → [FastAPI :8096 на Contabo]
[Oura Ring]   → Oura Webhook       →             [FastAPI :8096]
                                                         ↓
                                                   [SQLite → PostgreSQL]
                                                         ↓
                                               [Baseline Algorithm]
                                                         ↓
                                            [TG Bot Alert] → пользователю
```

### 4.3 Baseline-алгоритм (anomaly detection)
- 7-дневное скользящее среднее HRV для каждого пользователя
- Алерт при падении HRV > 20% от личного baseline
- Алерт при Readiness Score < 50 (Oura)
- Корреляция: ночная TG-активность + низкий HRV утром = high confidence alert

### 4.4 Мониторинг Telegram-активности (v2)
- Telethon (MTProto API): online-статус пользователя
- Детекция ночной активности (после 01:00)
- Требует: пользователь добавляет бота в контакты + не скрывает online-статус
- Polling каждые 15 минут — легальный паттерн при наличии согласия

### 4.5 Эскалация интервенций (ключевая фича)

| Уровень | Триггер | Действие | Задержка |
|---------|---------|----------|----------|
| 1. Бот | Аномалия в данных | Мягкое уведомление в TG | Мгновенно |
| 2. Повторный | Нет ответа 2ч | Повторное уведомление + рекомендация | +2ч |
| 3. IVR | Нет ответа 4ч при критическом HRV | Автоматический звонок (Voximplant) | +4ч |
| 4. Живой звонок | Нет ответа на IVR | Екатерина или ассистент звонит лично | +6ч |

**IVR (Interactive Voice Response):**
- Провайдер: **Voximplant** (российский, работает без санкционных ограничений)
- Стоимость: ~$0.02/звонок
- TTS на русском языке (встроенный + Яндекс SpeechKit)
- REST API для запуска из Python backend

### 4.6 Консультации с психологом

**Формат:** Екатерина — психотерапевт, в MentalStab работает в формате консультирования (мониторинг + поддержка). Если потребуется психотерапия — участники получают доступ на выгодных условиях. При необходимости медикаментозной поддержки — направление к психиатру или неврологу.

- Разовые сессии: 7 500 ₽ через Tribute
- Включённые в подписку: Co-Pilot — 2/мес, Mission Control — до 4/мес
- IFS-подход, 60 минут
- Calendly для записи: calendly.com/cher-k240596/meet-with-me

## 6. Unit Economics

| Метрика | Autopilot (4 900) | Co-Pilot (14 900) | Mission Control (39 900) |
|---------|-------------------|-------------------|------------------------|
| Выручка/мес | 4 900 | 14 900 | 39 900 |
| COGS психолог | 0 | 10 000 (2 сессии) | 20 000 (4 сессии) |
| COGS инфра + IVR | 700 | 1 000 | 2 000 |
| **Gross Profit** | **4 200** | **3 900** | **17 900** |
| **Gross Margin** | **86%** | **26%** | **45%** |
| LTV (8 мес avg) | 33 600 | 31 200 | 143 200 |
| CAC (целевой) | 3 000–5 000 | 3 000–5 000 | 3 000–5 000 |
| LTV/CAC | 8.4x | 7.8x | 35.8x |

**Bottleneck:** пропускная способность Екатерины — 25 клиентских часов/нед.
- Mission Control: до 25 клиентов
- Co-Pilot: до 50 клиентов
- Autopilot: не ограничен

**Реалистичный сценарий (год 1):** 30 Autopilot + 15 Co-Pilot + 5 MC = **569 500 ₽ MRR = 6.8 млн ₽ ARR**

**Масштабирование:** 2-3 ассоциированных психолога под супервизией Екатерины (год 2).

### Размер рынка
- **TAM:** ~13 млрд ₽/год (1.8 млн IT-специалистов в РФ, 15% готовы платить)
- **SAM:** ~1.2 млрд ₽/год (100K вайб-кодеров, 10% готовы платить)
- **SOM:** 29–72 млн ₽/год (200-500 подписчиков через прямые каналы)

## 9. Риски

| Риск | Уровень | Митигация |
|------|---------|-----------|
| **152-ФЗ** (health data = спецкатегория) | Высокий | Хранить в РФ (Selectel/Yandex Cloud). Явное письменное согласие. НЕ хранить на Contabo (Германия) |
| **Retention** (mental health apps: 3-4% на 30 день) | Высокий | Human touch + эскалация звонком = НЕ app. "Ты не отключаешь мониторинг сервера когда нет ошибок" |
| **Масштаб Кати** (single point of failure) | Критический | Год 1: ОК. Год 2: 2-3 ассоциированных психолога + "MentalStab method" |
| **Стигма "за мной следят"** | Средний | Только opt-in. Прозрачные настройки. Пользователь отключает любой источник |
| **Кризисные ситуации** (суицидальные мысли) | Критический | Протокол кризисного вмешательства. Горячая линия в боте. Направление к психиатру |
| **Юридическое** (НЕ медицина) | Средний | Позиционировать как "консультирование и мониторинг", НЕ "лечение" и НЕ "медпомощь" |
