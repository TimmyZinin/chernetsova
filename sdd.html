S9
FAQ

FAQ — Accordion

Реализация — native <details>/<summary> (CSS-only)
Что такое Mission Control?
Система проактивного мониторинга твоего психического состояния. Мы следим за HRV, сном, стрессом и ежедневными чекинами — и вмешиваемся до того, как станет плохо.
Зачем нужен трекер?
Объективные данные от трекера убирают слепое пятно: ты можешь не замечать накопленный стресс субъективно, но HRV не врёт.
Что если я не отвечу боту?
После 2 пропущенных чекинов система автоматически поднимает алерт и Екатерина видит пропуск. Мы напишем сами.
Это терапия?
Mission Control — это мониторинг и поддержка, не терапия. При необходимости мы рекомендуем или организуем полноценные терапевтические сессии.
Мои данные в безопасности?
Данные хранятся на серверах в России, зашифрованы, не передаются третьим лицам. Работаем по договору с NDA-клаузулой.
Могу ли я отменить подписку?
Да, в любой момент — без объяснений и штрафов. Один клик в боте или сообщение Екатерине.
⚙️
Техника: <details>/<summary> — нативный HTML, без JS. CSS transition на max-height для плавного открытия.

PF
Performance

Performance

Цели и ограничения
  • Total HTML file size: < 150KB target
  • Нет внешних JS-библиотек — zero bundle overhead
  • Lazy load below-fold content (IntersectionObserver)
  • Preconnect Google Fonts — устраняет latency при загрузке шрифтов
  • CSS-only анимации где возможно (GPU-accelerated transform/opacity)
  • will-change: transform — только для активно анимируемых элементов
  • prefers-reduced-motion — отключение анимаций по user preference
/* Respect reduced motion */ @media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }
