02

Целевая аудитория

Кто это
  • Вайб-кодеры (Claude Code, Cursor, Copilot)
  • IT-предприниматели, фаундеры, CTO
  • Возраст: 25–40 лет
  • Доход: 150K–450K ₽/мес
Боль
  • Выгорание
  • Потеря сна
  • Тревожность и FOMO
  • Разрушение отношений
Мотивация
  • Оставаться продуктивным
  • Не "сломаться" на дистанции
  • Понимать своё состояние
  • Сохранить отношения и здоровье
03

Конкурентный ландшафт

Прямых конкурентов: 0.  MentalStab — first mover в нише "проактивный мониторинг ментального здоровья IT-специалистов". Никто не занимает квадрант High-touch + Tech-specific.

Квадрант Продукты Слабость
High-touch + Tech-specific ПУСТО — мы здесь —
High-touch + General Yasno, BetterHelp Реактивная модель
Low-touch + General Calm, Headspace, Wysa Нет человека, нет мониторинга
04

Функциональные требования

4.1 Ежедневные чекины MVP
  • Telegram-бот спрашивает: настроение, сон, энергия, стресс 1–5
  • Свободное поле: "что произошло?"
  • AI-анализ паттернов — скользящее среднее 7 дней
4.2 Интеграция трекеров v2
Трекер Интеграция Стоимость
Oura Ring REST API v2, OAuth, webhooks $0
Apple Watch Health Auto Export $0
Terra API При 50+ юзерах $399/мес
Метрики: HRV, сон (стадии), ЧСС покоя, активность
4.3 Мониторинг Telegram-активности v2
  • Telethon: online-статус пользователя
  • Детекция ночной активности (после 01:00)
  • Корреляция с данными трекера
4.4 Эскалация интервенций Ключевая фича

4.5 Консультации с психологом
  • Разовые сессии через Tribute — 7 500 ₽
  • Включённые в подписку: Co-Pilot 2/мес, Mission Control до 4/мес
  • IFS-подход, 60 минут
